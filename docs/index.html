VPL Teacher Tools

Mac

Download the most recent file below, double-click its icon to mount the image, and copy VPL3Server.app (maybe shown as VPL3Server depending on your settings) to a convenient location on your Mac, such as the Applications folder or the Desktop.

At least the first time, the Mac is likely to complain that the application represents a security risk. To open it anyway, control-click its icon, choose Open in the contextual menu; in the dialog, click the checkbox to not show warnings again if there is one, and click Open. Repeat once if it fails.

Should it still fail, download the most recent generic version below, uncompress the zip file, and double-click launch_mac.command (maybe shown as launch_mac).

Windows

We don't have yet a packaged application for Windows. You should install Python 3 unless you already have it (we suggest the official distribution, but Anaconda is a popular alternative if you intend to use Python beyond VPL; if you're asked, check the checkbox Add Python 3.8 PATH).

Then download the most recent generic version below and uncompress the zip file. To start the server, double-click launch_windows.bat. It will download and install required libraries if needed.

Linux

Make sure Python 3 is installed (type python3 in a terminal, ctrl-d to quit). Download the last zip file for Windows above, uncompress it, open a terminal, cd to the root directory of the uncompressed zip file. The first time, type

python3 -m pip install websocket websockets qrcode

Then get the most recent generic version below. To start the server, type

python3 launch_tk.py

* Database format change. Please delete file vplserver-db.sqlite in your home directory manually. You will loose all its content, including the pupil names and the files. To preserve the pupil names, you can export them with your current version of the VPL Teacher Tools, delete vplserver-db.sqlite and upgrade the VPL Teacher Tools, and import the pupil names in the new version. Likewise for the teacher files, you can export them with the current version and reimport them with the new version.
