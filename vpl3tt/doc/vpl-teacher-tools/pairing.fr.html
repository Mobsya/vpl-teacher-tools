Élèves

Glissez-déposez dans un groupe pour constituer des groupes d'élèves.

Les élèves peuvent être ajoutés dans l'onglet Élèves.

Groupes

Glissez-déposez les élèves entre les groupes ou hors d'un groupe pour les réorganiser.

Sélectionnez un groupe avec un robot pour afficher le code de lancement de VPL avec les paramètres de connexion.

Robots

Glissez-déposez un robot sur un groupe pour l'associer au groupe.

L'accès aux robots se fait par le Thymio Device Manager (Thymio Suite).

L'accès aux robots se fait par le serveur VPL, sans logiciel supplémentaire.

Lancement de VPL spécifique à un groupe

Sélectionnez le groupe dans la vue «Groupes» ci-dessus.

Page de lancement unique

Cette page permet aux élèves de lancer VPL en cliquant sur leur groupe dans une liste. Elle est commune à tous les groupes.
